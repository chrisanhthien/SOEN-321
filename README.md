# Video sharing platforms: Risk of Misinformation and Personal Health Disclosures

This project analyzes TikTok video data to identify and quantify personal health disclosures. The analysis focuses on detecting health-related terms and self-identification language to understand the prevalence of such disclosures and their association with hashtags.

---

## Introduction

Social media platforms like TikTok have become spaces where users share personal experiences, including health-related disclosures. This project aims to:
- Identify videos containing health-related terms and self-identification language.
- Analyze hashtags associated with these disclosures.
- Quantify the prevalence of personal health disclosures in the dataset.

The results can provide insights into how users discuss health topics on TikTok.

---
## Purpose

The scripts in this repository are used to:
1. Detect health-related terms and self-identification language in TikTok video descriptions.
2. Analyze hashtags associated with health disclosures.
3. Calculate prevalence metrics for personal health disclosures.

---

## Usage

1. **Run the analysis scripts**:
   - Use `personal_data_disclosure_in_description.py` to analyze video descriptions.
   - Use `tiktok_data_filtering.ipynb` for data preprocessing.

2. **Input Data**:
   - Ensure `tiktok_data_medical_large.csv` is in the repository root.

3. **Outputs**:
   - Results are saved as CSV files for further analysis.

---

## Dataset

The analysis uses a dataset of TikTok videos stored in `tiktok_data_medical_large.csv`.
